# Text Formatting Challenge

Write a Java program that prints out the following patterns using loops:
```
1)
*
**
***
****
.........
2)
..........
****
***
**
*
3)
     *
    ***
   *****
  *******
...........
4)
............
  *******
   *****
    ***
     *
```

### Build & Run
```
make
./bin/formatting
```
